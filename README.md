## Installation

- Important: You must have installed the Node.js and Git

```bash
# clone it
$ git clone https://github.com/Wesley-AlvesRolim/NLW-Together-ReactJS.git

# Install dependencies
$ yarn

# Running script
$ yarn start
```

- And enter in your browser using http://localhost:3000
